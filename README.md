## ğŸ“ Tafseel (Description)

**HS TOOLKIT** ek modern aur responsive web dashboard hai, jise aapke rozana ke sabhi utility aur social media ke kaam ek hi jagah par manage karne ke liye design kiya gaya hai. Iska design bahut hi pur-kashish (attractive) gradient-themed hai aur isme Dark/Light Mode toggle ki sahulat bhi hai. Yeh project **Haseeb Sahil** ne banaya hai.

Is toolkit mein photo editing, social media downloaders, data management aur deegar zaroori tools ko khoobsurat, card-based layout mein tarteeb diya gaya hai.

## âœ¨ Khaas Khoobiyan (Key Features)

* **Modern UI/UX:** Zabardast CSS gradients, glassmorphism design (using `backdrop-filter`), aur ek saaf-suthra card-based layout.
* **Dark/Light Mode:** Theme badalne ke liye aasan toggle, taake aap apni marzi ka mode chun saken.
* **Responsive Design:** Mobile aur desktop, dono par behtareen dikhne ke liye optimize kiya gaya hai.
* **Tools ki Division (Categorization):**
    * **Main Utilities (`index.html`):** Jaise HS Database, WA & TT Stalker, Weather Check, TTS Generator, SMS Bomber, Age Calculator, Temp Mail/Num, aur Custom Mail.
    * **Photo Hub (`index2.html`):** Is mein AI Image Tool, Face Swap, Image Editor, Image Generator, Image Enhancer, aur BG Remover shamil hain.
    * **Social Hub (`social-tools.html`):** Google Drive, YouTube/TikTok Video Downloaders, aur Instagram/Facebook ke liye alag tools.
* **Developer Info:** Developer (`Haseeb Sahil`) se raabta (contact) karne ke liye ek alag page (`info.html`).
